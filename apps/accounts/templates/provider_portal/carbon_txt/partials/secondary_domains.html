Your secondary domains are the domains of your customers, or any other domain which you want to be attributed to you in the Green Web platform.

In order to attribute these domains correctly to your provider, we need to link them to your primary domain's carbon.txt file. You can do this by returning a special HTTP response header, or setting up a DNS TXT record.

  • Use an HTTP Header
  • Use a DNS TXT record
Link this domain to an existing carbon.txt file using HTTP Headers

For providers who resell other services, we recommend doing this by setting up a custom HTTP response header on all the domains you host, linking to the carbon.txt hosted on your primary domain.

Configure your web server or proxy to add the CarbonTxt-Location header listed below (see the carbon.txt documentation for further details, and instructions for some popular web servers such as Apache, Nginx and Caddy).
